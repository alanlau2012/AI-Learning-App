# AI 应用工程学习 APP

面向企业 AI 应用负责人的交互式学习 Web 应用（56 讲）。纯前端、内容数据驱动。

## 开发

```bash
npm install        # 安装依赖
npm run dev        # 本地开发
npm run build      # 生产构建
npm run typecheck  # TypeScript 类型检查
npm run lint       # ESLint
```
